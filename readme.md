# MVP API Template #
## API Template For MVPs Built With Laravel & GraphQL ##

### What's Included ###

- Docker development configuration
- Stripped of unnecessary views/frontend code
- Lighthouse GraphQL
- Tymon JWT authentication
- Laravel CORS
- Authentication, email verification, password reset
